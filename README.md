# Card-notification

## Описание проекта

Приложение, реализующее автоматическую рассылку уведомлений клиентам банка о скором истечении срока действия их банковских карт.

## Функциональность

*   **Управление клиентами:**
    *   Создание нового клиента с защитой от дубликатов.
*   **Управление банковскими картами:**
    *   Выпуск новой карты для существующего клиента с проверкой уникальности номера.
    *   Аннулирование (закрытие) действующей карты клиента.
*   **Уведомления по расписанию:**
    *   Отправка уведомлениий по расписанию (либо таймеру).
    *   Поиск всех карт, срок действия которых истекает (например, через неделю).
    *   Генерация уведомления с записью в файл (позже с отправкой на почту).

## Стек

*   **Язык:** Java 17
*   **Сборка:** Maven
*   **База данных:** Реализована абстракция, что позволяет легко менять источник данных.
    *   **Базовая реализация:** HashMap.
    *   **SQL:** Spring data.
